Design UI UX Web

Gambaran kasar mengenai perancangan UI UX Web untuk pengumpulan tugas mata kuliah Elektronika selanjutnya.

UI UX design adalah proses merancang antarmuka pengguna (UI) dan pengalaman pengguna (UX) pada sebuah produk atau aplikasi. Kami diminta oleh dosen Elektronika kami, Sandy Bhawana Mulia, S.Pd., M.T. untuk membuat website yang selanjutnya akan digunakan untuk pengumpulan tugas dan project kami untuk satu semester kedepan.

Pada gambar diatas merupakan gambaran dan konsep kasar yang telah saya buat untuk UI UX Website saya. Konsep kasar yang telah saya buat disempurnakan dan dioptimalkan fitur dan tampilannya agar memudahkan saya maupun yang mengakses.

Ada beberapa fitur penting yang saya tambahkan pada website yang saya buat, seperti navigasi pada header agar memudahkan para pengguna berselancar di website yang saya buat. Saya juga menambahkan laman Contacts agar dosen atau pengakses dapat menghubungi saya melalui WhatsApp, Instagram, dan Telegram.
